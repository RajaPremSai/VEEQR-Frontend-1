# Product Overview

VEEQR is a role-based Vehicle Entry/Exit QR system built as a React web application.

## Core Functionality

- QR code-based vehicle entry/exit tracking
- Role-based access control with three user types:
  - **USER**: Vehicle owners who can manage their vehicles and view logs
  - **SECURITY_GUARD**: Gate operators who scan QR codes and monitor entry/exit
  - **MANAGER**: Administrators who manage users, guards, vehicles, and system settings

## Key Features

- Vehicle registration and management
- QR code scanning for entry/exit
- Real-time logging and monitoring
- Announcements system
- Gate management
- User and security guard administration

## Authentication

- JWT-based authentication
- Role-based routing and access control
- Default manager credentials: manager@example.edu / StrongPass@123
